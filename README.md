## Maya Curve to Joint Tool
A simple tool created to generate a desired number of joints along the length
of a curve drawn in Autodesk Maya.

This was done for a college assignment in January 2018.

### Dependencies

This tool relies on a version of Qt.py (included in repository) to function.

Credit to mottosso for his work, and for future versions see the link below:

https://github.com/mottosso/Qt.py
